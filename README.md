# BetterSaves
Adds support for *infinite* save files (or however many your computer can handle) to DELTARUNE

Includes some extra QoL features alongside this, such as save file icons that give quick information about a file.
## [Read more here on the gamebanana page!](https://gamebanana.com/mods/619707)

Source code provided as [UTMT](https://github.com/UnderminersTeam/UndertaleModTool) project files. 

Before applying a chapter-specific project file, make sure to apply the *_everychapter* project as well. This includes the chapter select project!
